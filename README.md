# Snake

贪吃蛇怎么翻比较好听呢？

## 说明

- 因为本人贼菜还不会写前端也不知道怎么做动画，所以只能在控制台里一帧一帧显示……

## 项目背景

贪吃蛇是大伙的佬朋友了就8用多说了8

## 开发环境

- Windows 10 64位 ( DirectX 12 )

## 功能

- 单次运行程序可以连续玩多局游戏
- 为了提高可玩性，蛇向上下左右移动时棋盘颜色分别显示白青紫黄四种不同颜色
